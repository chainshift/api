# CBW API手册
  本文档对CBW的租户接入和相应的API进行说明。
  
## 接入步骤
- 向CBW运营方提供开户信息：负责人手机号码，邮箱地址，接入系统名称，接入系统回调地址。
- CBW运营方审核通过开户完成后，向接入方提供tenantApiKey，请妥善保管，防止泄漏和遗失。
